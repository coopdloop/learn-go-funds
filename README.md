# Running

## Setup

```
$ git clone https://github.com/coopdloop/learn-go-funds
```

## Running

### Example 1

```
$ go run startme.go -concurrent0
```

### Example 2

```
$ go run startme.go -concurrent2 -problem
```

### Example 3

```
$ go run startme.go -concurrent2 -solution
```
